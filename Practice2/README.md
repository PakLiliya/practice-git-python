# Practice 2: Python Control Flow Examples

## Project Overview
This project contains Python examples covering fundamental control flow concepts, created based on the W3Schools Python tutorial. Each example file demonstrates specific programming concepts with practical code snippets.

## File Contents Description

### Boolean Examples (`boolean/`)
1. **boolean_intro.py** - Basic boolean values (True/False) and their usage
2. **boolean_comparison.py** - Comparison operations (==, !=, >, <, >=, <=)
3. **boolean_operators.py** - Logical operators (and, or, not) with examples

### Conditional Statements (`if_else/`)
1. **if_statement.py** - Basic if statements for conditional execution
2. **if_else.py** - If-else constructs for binary decisions
3. **if_elif_else.py** - Multiple condition checking with elif
4. **short_hand_if_else.py** - Ternary conditional expressions
5. **switch_if_elif_else.py** - Implementing switch-like functionality using if-elif chains

### Loop Constructs (`loops/`)
1. **while_loops.py** - Basic while loop implementation
2. **while_loop_break.py** - Using break statement to exit while loops
3. **while_loop_continue.py** - Using continue statement to skip iterations
4. **for_loops.py** - Basic for loop with iterables
5. **for_loop_break.py** - Using break statement to exit for loops
6. **for_loop_continue.py** - Using continue statement in for loops
