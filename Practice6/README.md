## Description

This practice demonstrates how to work with files, directories, and Python built-in functions.
Examples are based on materials from W3Schools.

The project includes examples of:

* Reading files
* Writing and appending files
* Copying and deleting files
* Creating and managing directories
* Using common built-in Python functions
