# Practice 5 – Python Regular Expressions

## 📌 Description
This project contains:
- Python RegEx exercises
- Receipt parsing using regular expressions

## 📂 Files
- regex_exercises.py – RegEx practice tasks
- receipt_parser.py – Receipt parsing program
- raw.txt – Raw receipt data

## 🔧 Technologies
- Python 3
- re module
- JSON

## ▶ How to Run

Run RegEx exercises:
python regex_exercises.py

Run receipt parser:
python receipt_parser.py

## ✅ Features
- Demonstrates re.search, re.findall, re.sub, re.split
- Uses metacharacters and quantifiers
- Extracts products, prices, total, date, payment method
- Outputs structured JSON
